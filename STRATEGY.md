# Vercel AI Edge HQ - Strategy

Edge-native TypeScript AI app runtime for low-latency agent UX

## Differentiator
This repository is intentionally differentiated from the rest of the Kazi SDK portfolio through framework-native architecture choices and release strategy.

- Keep a single-framework focus per repository.
- Preserve a shared mission-routing contract.
- Ship with deployment and verification defaults.

- Time-to-first-run under 10 minutes.
- Deterministic smoke checks in CI.
- Deployment readiness with documented secrets and rollback path.
