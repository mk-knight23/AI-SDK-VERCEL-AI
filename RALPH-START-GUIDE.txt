## üõë HOW TO STOP

Press Ctrl+C (graceful shutdown)
OR: pkill -f 'ralph --prompt'

Ralph saves state and can be resumed!

## üîÑ RESUME AFTER INTERRUPTION

Just run ./start-ralph.sh again
Ralph auto-resumes from where it left off

## üÜò TROUBLESHOOTING

"tmux not installed": brew install tmux
"Ralph not found": Install from https://github.com/frankbria/ralph-claude-code
"API key errors": Ralph will pause, rotate key, Ralph auto-resumes
"Network outage": Ralph pauses affected projects, continues with others
